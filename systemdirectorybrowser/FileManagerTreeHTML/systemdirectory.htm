Project: File System Browser for FLTK 1.1.x


At the end I have plesure to announce the new FLTK widget. A powerfull widget which could be used for many projects (System Directory Browser).
I have been working on the project for several months .. since my job take most of my time, unfortunatly I could not release it before.

with this package I will not release any API documentation. Since it needs to be writen and I did not write it yet. The second reasen behinde not writing the API is waiting for further comments from users. maybe bugs .. whatever ... I will be waiting a litile bit for the comment, and then I will be writing the API hopefully it will not take a long time.

For using the widget ... simply write
Fl_SystemDirecotry_Browser *br= new Fl_SystemDirecotry_Browser(x,y,w,h,l)
x: is start point of X
y: is start point of Y
w: is the width
h: is the hight
l: is the lable

Please feel free to use it, feel free to send comments, and feel free to send bugs... I hope that it will be usefull for you to use it.

I would like to thanks Don Zimmer for his greatefull help in doing this widget. As the base tree browser was from him, and all other FLTK developer whow helped me .. many thanks..
